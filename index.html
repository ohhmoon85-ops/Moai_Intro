㈜ 모아이 | 2025.8.28

🧠 지능형 인덕션

기존의 정적인 제어를 넘어선 AI 기반 동적 제어가 핵심입니다.

  • 정밀한 환경 인지: 접촉/비접촉 센서로 용기 바닥 온도를 실시간 측정
  • 다중 센서 분석: 온도 변화율 패턴으로 용기 위치, 크기, 끓어넘침까지 감지
  • 생성형 AI 모델: 예측 불가능한 상황에 실시간으로 새로운 제어 로직 생성
  • 냄새 발생 차단: 과열과 탄화를 사전에 방지하여 냄새 원인 제거
특허 현황:
• 온도조절 인덕션 레인지 (등록: 10-2708883)
• 스마트쿠킹 장치 (출원: 10-2025-0104924)

🌪️ ValleyFlow™ 후드

360도 전방향 포집과 자동 승하강으로 완벽한 조리부산물 제거

  • 강력한 공기증폭: 적은 에너지로 주변 공기를 강력하게 흡입
  • 스마트 연동: 인덕션의 조리 강도에 맞춰 풍량 자동 조절
  • 안전한 설계: 팬이 조리부산물에 노출되지 않아 화재 위험 제거
  • 공간 효율성: 조리 시작/종료에 따른 자동 승하강
특허 출원:
• ValleyFlow™ 기술 (10-2024-0175859)
• 승하강 후드 시스템 (10-2025-0058816)

🔄 현재 기술의 한계와 혁신적 해결책

현재 기술의 문제점

  • 정적인 제어로 돌발 상황 대응 불가
  • 간접적 온도 감지로 정밀도 부족
  • 제한적 자동화 (물 끓임, HMR 등만)
  • 수동적 후드 작동으로 효율성 저하
  • 냄새 발생 후 처리하는 방식

혁신적 해결책

  • AI 실시간 분석 복합적 조리 환경 인지
  • 생성형 모델 예측 불가능한 상황 대응
  • 동적 제어 스스로 최적 제어 방식 판단
  • 지능형 연동 인덕션-후드 자동 협업
  • 사전 차단 냄새 발생 원인부터 제거
